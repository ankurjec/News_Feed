*******************Usage And Installation Of the Application*******************


1. Goto https://github.com/ankurjec/News_Feed/tree/master and Download in zip.

2. Extract the folder i.e 'News_Feed-master', rename it to 'News_Feed' and then place the folder  in
 XMPP htdocs folder.

3. Now type the url localhost/News_Feed in browser and the
 Application will start running.
